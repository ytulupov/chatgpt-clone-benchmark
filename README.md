# ChatGPT UI Clone

Create a ChatGPT-like interface using React, TypeScript, and Tailwind CSS.

## Tech Stack
- React 18
- TypeScript
- Tailwind CSS
- Vite

## Requirements
1. Dark theme matching ChatGPT's design
2. Sidebar with chat history and "New Chat" button
3. Main chat area with message bubbles (user/assistant)
4. Fixed input area at bottom with textarea and send button
5. Mock AI responses (simulate typing effect)
6. Responsive design (mobile-friendly)
7. Smooth animations

## Getting Started
```bash
npm install
npm run dev
```
